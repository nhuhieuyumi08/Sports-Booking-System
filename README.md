# Sports Booking System

## Project Overview
This project analyzes a Sports Field Booking System that allows users to search, book, and manage sports field reservations online.

## My Role
Business Analyst

## Objectives
- Analyze business requirements
- Design system processes & UI/UX
- Create system diagrams
- Define user stories and use cases
- Test UAT

## Main Features
- User registration and login
- Search sports fields
- Book a field
- Payment integration
- Tournament management
- Ranking system
- User feedback
